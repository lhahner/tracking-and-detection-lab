Edozie et. al \cite{edozie} provides a review across the past developments of object detection techniques in the past 26 years. Besides describing the evolution of deep learning based object detection systems they also compare them to the classical non-deep-learning based object detection methods and algorithms.

% Why is this interesting in this report?
They provide a wide overview of datasets, methods and research that was important in the past 26 years. Beside providing classical object detection methods they also present an overview of modern deep learning based object detector which should be summarized here to give an impression of the state of the art.

% What review is provided
In their work they present different ways to categorize object detection systems.
One way object detection systems are separated in One-Stage and Two-Stage object detectors. Where the two-stage detector first generates proposals of the detection and then encodes them to make class predictions. Otherwise the first-stage detector doesn't consider region proposals, they simply rely on the locations with the most objects and either classify them as an object or background. Through this simplification One-Stage detectors are generally faster than two-stage detectors.

This categorization approach leaves out other architectures like transformer-based models like DETR \cite{} and Multi-path CNNs like PAN \cite{}. Instead they summarize the different detection architectures by the categories of tradional detectors which include non deep learning methods, conventional deep learning methods that only inlcude two-stage detection systems, Hybrid\/Briding which provides a category for Multi-path detection systems and advanced deep learning which is a category for one-stage detectors and transformer based models.
